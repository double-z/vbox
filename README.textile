h1. VBox

Vbox is a web-based front-end for management of VirtualBox on a headless server.

I've only just started making it, and for the moment I'm only concerned with the
features that I personally want. I'd be happy for others to contribute.

VBox is written in Rails.

h2. Immediately Planned Features

* Connecting/unconnecting hard disks to machines
* Cloning hard disks
* Cloning machines
* Distinguishing between the different disk types
